# docker_compose_demos
Example of communication between services in two docker compose

## Bringup redis via one compose
```cd service2```\
```docker-compose up```

## Bringup flask app via another compose
```cd service1```\
```docker-compose up```

## Access url
[http://127.0.0.1:5000/](http://127.0.0.1:5000/)
